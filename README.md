## VMC4UE とは
VirtualMotionCapture から OSC で姿勢情報を受信してUEに反映するためのプラグインです。

動作には sh_akira さんの VirtualMotionCapture(v0.36～) が必要です。
https://sh-akira.github.io/VirtualMotionCapture/download.html

※それ以前のバージョン(～v0.35)では使えません

## 動作環境
- UE4.21
